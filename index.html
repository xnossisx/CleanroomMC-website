About Us

Our mission is to provide a set of mods, coremods, APIs, and guides to incubate the creation of Minecraft mods/modpacks on 1.12.2. Contributions and additions are welcome!

Projects

Check out our Github to see all projects.

AssetMover (link)

Obtains vanilla/mod assets at runtime by downloading from URLs. This avoids needing to include them in builds, potentially violating licenses.

Millennium (link)

Ports various features from post-1.12.2 versions of Minecraft.

Better Questing (link)

Continues this quest mod from where @Funwayguy stopped development, adding bugfixes and improvements.

Encyclopedia (link)

A guide for all 1.12.2 Forge modders (Currently a work in progress)!

OrangeCore (link)

Adds many custom food mechanics, building on AppleCore. Inspired by TerraFirmaCraft's food system.

Template Dev Environment (link)

A simple template for starting a 1.12.2 mod project.
